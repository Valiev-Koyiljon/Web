# Investigating a Reinforcement Learning Framework for Smart Grid Energy Management

**Author:** Koyilbek Valiev  
**Venue:** Journal of the Korea Institute of Communication Sciences (JKICS), Vol. 51, Issue 2, 2026  
**Status:** Published

## Abstract

This paper investigates a reinforcement learning framework for smart grid energy management, covering market trading, demand forecasting, and vertical agent integration.
