# A Study of Reinforcement Learning Framework for Energy Management in Smart Grids

**Authors:** Koyiljon Valiev, Sukhrob Ikromov, Kim Young-il  
**Venue:** Journal of the Korea Institute of Communication Sciences (KICS)  
**Volume:** Vol. 51, No. 2, pp. 324–347, February 2026  
**DOI:** [10.7840/kics.2026.51.2.324](https://doi.org/10.7840/kics.2026.51.2.324)  
**Status:** Published

## Abstract

This paper investigates a reinforcement learning framework for smart grid energy management, integrating market trading, load forecasting, and vertical agent architectures. The proposed system enables autonomous decision-making for energy buying, selling, and storage operations in dynamic grid environments.

## Key Contributions

- Reinforcement learning agent for real-time energy market trading
- Load forecasting module integrated into the RL reward structure
- Vertical agent hierarchy enabling multi-level grid control
- Evaluation on real-world smart grid datasets
